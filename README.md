## What is Farmore?

> *Farmore is a chain-neutral protocol for unified blockchain identity, with a wallet that gives you one name, one account, and one balance across every blockchain.*

Register once, pick a handle (e.g. `@nododile`), and see one balance across every chain. Send by handle, not by 40-character address. No bridges to choose, no gas token to hold, no seed phrase to lose.

**FAR** is the network's token. It bonds the solvers that route cross-chain transfers, funds a safety pool, and is earned only through verified, useful activity.

## At a glance

- **Three promises** — one name, one balance, one account
- **Self-custodial** — passkey / biometric setup, no seed phrase to lose
- **Chain-neutral core + swappable adapters** - new chains plug in without touching the core
- **Threshold signing (MPC)** — one logical account, valid signatures on every chain
- **Solver-based routing** — competing solvers fulfil intents, bonded in FAR
- **Fixed supply** — 1,000,000,000 FAR, released only as real activity earns it
- **70% community** allocation;
